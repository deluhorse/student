# WM-B2C

基本python(version >= 2.7)的saas平台

## 信赖包

- Tornado-MySQL >= 0.5.1
- futures >= 3.1.1
- mysql-python >= 1.2.5
- pycrypto >= 2.6.1
- qiniu >= 7.1.2
- redis >= 2.10.5
- rsa >= 3.4.2
- tornado >= 4.5.1
- xmltodict >= 0.11.0

## windows安装mysql-python
需先安装：

- Microsoft Visual C++ 9.0
- mysql-connector-c-6.0.2-win32

## 启动服务
